# 🖥️ Atividade Prática – Montagem de Rede Classe C
## 🎯 Objetivo

Montar e configurar uma rede local (LAN) utilizando endereçamento IPv4 Classe C no Cisco Packet Tracer, aplicando corretamente os conceitos de:

* Endereçamento IP
* Máscara de sub-rede
* Comunicação em rede local
* Teste de conectividade (Ping)

## 📌 Situação-Problema

Uma pequena empresa precisa configurar sua rede interna para permitir a comunicação entre seus computadores.

Você foi contratado(a) para montar essa infraestrutura no Packet Tracer.

A rede deverá utilizar a seguinte faixa de endereçamento:

```text
Rede: 192.168.10.0
Máscara: 255.255.255.0
```

## Requisitos da Atividade

Você deverá montar uma topologia contendo:

* ✅ 1 Switch
* ✅ 6 PCs
* ✅ 1 Notebook
* ✅ Cabos de rede adequados

## Configuração

Todos os dispositivos devem pertencer à mesma rede.

Utilize endereços válidos dentro da faixa:

```text
192.168.10.X
```


# Testes Obrigatórios

Após a configuração:

Realizar testes de conectividade utilizando o comando:

```text
ping
```

Testar a comunicação entre:
* PC → PC
* PC → Notebook
* Notebook → PC
